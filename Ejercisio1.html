Etiqueta body

Es el cuerpo de la pagina web como lo Titulos, Parrafos de texto, Imagenes, Videos etc.

Funcion de la etiqueta head

Su funcion es para poner el nombre del encabezado "Como aqui se muestra en esta paguina que se llama Etiquetas"

(h1) Esta etiqueta hace un titulo como se muestra aqui

(h2) Pone un subtitulo como se muestra aqui

Etiqueta (P)

(p) Pone solo texto por default sin ningun diseño o color

Etiqueta (hr /)

La etiqueta hace una separacion de texto con una linea

Etiqueta (i)

La etiqueta (i) muestra las palabras en forma italica Ejemplo: Tecnologias de la informacion

Etiqueta br

Es un salto de linea Ejemplo:

Universidad Politecnica de Tlaxcala

La etiqueta img

La etiqueta sirve para poner una imagen, siempre y cuando este en la misma carpeta o seleccionar la ruta de la imagen, y como extra se puede porner la altura y lo ancho de la imagen

La etiqueta embed

La etiqueta embed inserta un video como el que se muestra

La etiqueta button

Representa un elemento cliqueable de tipo botón que puede ser utilizado en formularios o en cualquier parte de la página que necesite un botón estándar como el que se muestra acontinuacion
